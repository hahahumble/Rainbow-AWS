# 🌈 Rainbow AWS: AWS console header color

Rainbow AWS is a Chrome extension that let you **change the color of the AWS console's headers based on the region you select**.

<img src="https://image-hosting-bucket.s3.us-west-2.amazonaws.com/images/rainbow-aws.png" alt="rainbow-aws" width="500px"/>

## 🚀 Links

* [GitHub](https://github.com/hahahumble/Rainbow-AWS)

## ✨ Features
* 🔥 Support the latest AWS console UI
* 🌈 Easy to use popup menu to quickly change the color
* 🎨 Use the palette to quickly choose the color you need

## ❗️ Notice
* This extension required the "storage" privilege to store the color you choose. It won't collect, read or transmit any data of any kind.

## 🔭 Development

If you want to develop Rainbow AWS, its super easy!

- Clone this repo
- `yarn install`
- `yarn start`
- Load the `build` folder in Chrome extension page
